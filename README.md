# Smart-Noise-Cancellation-System
In many practical electronic and communication systems, signals are often affected by unwanted disturbances called noise. Noise can distort the

original signal and reduce the quality of communication or audio signals. Therefore, it is important to reduce or eliminate noise from signals.

The proposed Smart Noise Cancellation System is designed using operational amplifiers and transistor amplifier circuits to demonstrate the basic principle of noise reduction in analog systems.

Initially, the summing amplifier using IC 741 is used to combine two signals: the desired signal and the unwanted noise signal. The output of this stage contains both components.

Next, the combined signal is passed through an active low-pass filter, which allows low-frequency components (useful signal) to pass while attenuating high-frequency noise components. This filtering process improves the quality of the signal.

Finally, the filtered signal is fed into a transistor amplifier stage using BC547 transistor. This stage increases the amplitude of the signal so that it can be clearly observed and used by other circuits.

Thus, by combining the functions of summing amplification, active filtering, and transistor amplification, the system effectively demonstrates how noise can be reduced and signal quality improved in analog circuits.
